# Electricity Consumption Dashboard

This folder contains all files related to the **interactive dashboard** for forecasting electricity consumption in IST North Tower.

## Files Overview

- `assets/` – Contains CSS styles
- `dashboard.py` – Main Python script
- `BT_model.sav` – Bagging trained model
- `NN_model.sav` – Neural Network trained model
- `XGB_model.sav` – XGBoost trained model
- `IST_North_Tower_2019_raw.csv` – Raw 2019 electricity consumption data
- `IST_North_Tower_Clean.csv` – Cleaned 2017–2018 data for reference
- `IST_North_Tower_test.csv` – Test dataset for 2019 predictions
- `requirements.txt` – List of Python libraries needed to run the dashboard
